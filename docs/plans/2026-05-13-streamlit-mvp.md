# MVP 设计：Streamlit 本地 GUI

## 本质

`wx-deal-radar` 是一个本地消费情报过滤器。它不替代微信，也不做自动交易，只把高噪声购物群里的消息流压缩成少量可行动提醒。

## 数据流

```text
wx-cli new-messages
  -> 目标群过滤
  -> 黑名单过滤
  -> 全局关键词 / 商品规则
  -> 价格提取与阈值判断
  -> Windows 弹窗 + JSONL 日志 + GUI 查看
```

## 第一版边界

第一版只支持本机 Streamlit GUI，不做账号系统，不做云同步，不做远程推送。微信小程序提醒、企业微信/飞书机器人、邮件提醒可以作为后续通知适配器。
